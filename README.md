# Portable Solar Power Bank System

## Overview
This project is a portable solar-powered power bank system designed to provide an alternative power source for electronic devices during outdoor activities such as camping, hiking, and traveling.  

The system combines Lithium battery storage and solar panel charging to create a portable and environmentally friendly charging solution.

## Features
- Portable power supply for smartphones and small electronic devices  
- Dual charging source:
  - Lithium battery pack
  - Solar panel charging
- Stable 5V USB output using step-up converter  
- Rechargeable battery system  
- Lightweight and suitable for outdoor usage  

## Tech Stack
- Power Electronics  
- Hardware Prototyping  
- Battery Management System  
- DC-DC Step-Up Converter  

## Hardware Components
- 6× Lithium Batteries (3.7V, 1500mAh)  
- Solar Panel (5V, 400mA, 2W)  
- H961-U Charging Module  
- 5V Step-Up Converter  
- Battery Holder (Parallel Configuration)  
- Jumper Wires  

## System Workflow
1. Lithium batteries store electrical energy  
2. Solar panel acts as alternative charging source  
3. Step-up converter regulates voltage to stable 5V output  
4. Devices can be charged via USB output  
5. Solar panel can recharge battery system during outdoor use
